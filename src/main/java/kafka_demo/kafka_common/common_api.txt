org.apache.kafka.common
interface
1. ClusterResourceListener 返回metadata在集群中更改的通知
    通过配置选项kafka.server:type=KafkaServer，name=ClusterId就可以使用cluster id，
    同时cluster id会作为metadata返回信息的一部分回馈给客户端。
    Serializers，client interceptors 和metric reportes可以通过ClusterResourceListener接口获取cluster id。
    方法：onUpdate(clusterResource ) 获取ClusterResource的更新
2. Configurable 由反射和配置参数实例化
    子接口ConsumerInterceptor<K,V>, Partitioner, ProducerInterceptor<K,V>
    方法：configure(Map<String,?> configs)
3.Metric 一个跟踪监测的数值度量
    方法：metricName()
            value()
